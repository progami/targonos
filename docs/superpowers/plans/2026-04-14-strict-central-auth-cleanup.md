# Strict Central Auth Cleanup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove the remaining dev auth bypass/default paths so every app and the portal rely on explicit central-auth configuration and portal-issued claims only.

**Architecture:** Tighten the shared `@targon/auth` contract first, then remove every portal and child-app shortcut that still bypasses it. Update the local SSO tests to inject a signed portal session instead of using a credentials backdoor.

**Tech Stack:** Next.js 16, NextAuth v5 beta, TypeScript, tsx/node test, Playwright
